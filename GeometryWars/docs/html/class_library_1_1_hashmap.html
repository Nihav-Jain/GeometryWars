Detailed Description

template<typename TKey, typename TData, typename HashFunctor = DefaultHash<TKey>>
class Library::Hashmap< TKey, TData, HashFunctor >

Templated class for an associated container TKey is the data type of the key to be used to store the data TData is the data type of data to be stored in this container (optional) HashFunctor is the functor type which defines the default hash functions.
